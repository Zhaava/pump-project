# Диагностика состояния насосного оборудования с помощью ML

Проект посвящен прогнозированию и диагностике состояния насосного оборудования для нефтяных скважин с применением современных методов машинного обучения.

## Основные технологии

В проекте используются и сравниваются следующие модели и подходы:
- **TCN** (Temporal Convolutional Network)
- **Transformer**
- **N-BEATS**
- **LAMA**
- **AutoML**
